# Super Castle Game

## Controls

* Arrow keys or WASD
* Tap on the screen
* Gamepad left analog stick (and B to restart the level)

## Installation

```bash
cd super2023
python3 -m venv virtual
. ./virtual/bin/activate
python -m pip install -U pip
pip install -r requirements.txt
yarn
```

## How to build

```bash
./build.sh
open build/index.html
```
